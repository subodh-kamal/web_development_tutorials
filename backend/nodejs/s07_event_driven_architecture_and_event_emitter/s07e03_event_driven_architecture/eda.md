# Event-Driven Architecture(EDA)

- Node.js is based on Event-Driven Architecture.
- ***Event-Driven Architecture*** is a design pattern to build a software.
- Every operations gets done after performing events.

**Examples of EDA Pattern**
1. Node.js
2. Browser DOM Events
3. Chat Apps
4. OS Handling User Inputs
5. YouTube Notifications
6. Payment Processing Systems
7. Automation Services -> For ex: (zapier.com) (pabbly.com)

---

**3 components of Event-Driven Architecture.**
1. Event Emitter
2. Event Listener
3. Event Handler
