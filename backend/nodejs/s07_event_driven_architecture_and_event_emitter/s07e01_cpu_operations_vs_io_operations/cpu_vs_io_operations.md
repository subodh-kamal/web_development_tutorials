- Node.js performs the I/O operations in asynchronouos fashion.

### CPU Operations: (CPU Bound Tasks)

- It uses main thread.

1. Arithmetic Operations.
2. Sorting.
3. Searching.
4. Rendering Graphics.

### I/O Operations: (I/O Bound Tasks)

- It usually takes time to perform actions.

1. File Reading & Writing.
2. DB Querying.
3. Network Request.
4. User Input.
