Dialog Boxes in jS

Dialog boxes is used to show different types of pop-ups to user.

There are three types of Dialog boxes in jS

  1. alert("Your message goes here.")

    The alert() function in JavaScript does not return any value, so its return value is undefined. This behavior is by design since alert() is used to display a message to the user, not to compute or return a value.

    Explanation:

    • The alert() function simply displays a dialog box with the specified message and an "OK" button.

    • After the user dismisses the dialog box by clicking "OK," the function ends, and since it doesn't explicitly return anything, JavaScript assigns it a return value of undefined.

  2. prompt("Your message goes here.")

  3. confirm("Your message goes here.")
