Data Types in jS

There are 2 types of Data Types in javaScript:

  • Primitive (also referred as 'value types')
  • Non-Primitive (also referred as 'reference types')

Primitive Data Types are further divided into 7 categories:

  1. number
  2. string
  3. boolean
  4. undefined
  5. null
  6. bigInt
  7. symbol

There is only one Non-Primitive Data Type and that is:

  • object

object can further be categorized into three types:

  1. object literals
  2. arrays
  3. functions
