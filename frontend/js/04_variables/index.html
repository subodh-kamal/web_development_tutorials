Variables in javaScript

A javaScript variable is just a name of storage location.

And we can assign that name in three ways:

  1. Using let keyword
  2. Using const keyword
  3. Using var keyword

Rules for Naming Variables:

A variable name can contain any of the following characters but not start with the digits.

  • Capital Letters: A to Z
  • Small Letters: a to z
  • Digits: 0 to 9 (but we cannot start variable name with digits.)
  • Only two special characters that are: Dollor($) and Underscore(_)
